## 📦 Sobre o Projeto

Aplicação simples de lista de tarefas (To Do List), criada com Angular CLI.  
Permite adicionar, listar e remover tarefas de forma prática.

---

## 🚀 Tecnologias

- Angular 17
- TypeScript
- HTML5 e SASS

---

## ▶️ Como Executar

### 1. Instale as dependências:

```bash
npm install
npm start
```
